This Windows 8.1 App demonstrates how the secondary command bar experience can be tweaked according the devices' capabilities.
The subject is a command bar with specific commands for an item in a listview.

---
Visual Studio 2013 requires the Behavior SDK: https://msdn.microsoft.com/en-us/library/windows/apps/dn457340.aspx.
In Visual Studio 2015 RC this is preinstalled. It suffices to add the reference.
